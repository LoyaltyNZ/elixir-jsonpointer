ODGN JSONPointer
=================

An implementation of RFC 6901 with inspiration from https://github.com/manuelstofer/json-pointer.git


## Installation

Add a dependecy to your project `mix.exs`:

```Elixir

def deps do
  [{:odgn_json_pointer, "-> 1.0.0"}]
end

```
